Criar Utilizador Autenticação.Gov

Para usar o MyMonitor é necessário ter um utilizador Autenticação.Gov (utilizador.Gov).

Os Utilizadores são uma espécie de conta que agrega sítios Web para serem acedidos por um grupo de pessoas.

Com o MyMonitor o processo passou a ser individualizado uma vez que a autenticação é individual. Assim, é necessário que cada utilizador.Gov aceda ao seu utilizador/conta ou grupo de sítios web de uma conta.
