Criar Categoria

você está em: criar categoria

Uma Categoria é um agregador de sítios web. As Categorias têm por objetivo agregar os sítios web por classes. São exemplos de classes: área governativa da educação, área governativa da economia, administração pública central, escolas, institutos públics, etc. Desta forma é possível organizar a visualização da informação no AdminMonitorSuite por listas de sítios web que estão, de alguma forma, relacionados e observar estatísiticas globais dessas agregações. Uma categoria é muito similar ao classificador "tag" que se usa vulgarmente nos blogs.

Nota: as Categorias não serão publicadas no Observatório público. No Observatório público os sítios web são publicados por via das Diretorias aos quais pertencem. As Categorias ajudam a produzir e organizar os Diretórios.
